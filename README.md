# Beagle Scraper

Ecommerce and product categories scraper developed in Python and BeautifoulSoup4

## Getting Started

Beagle Scraper requires a machine with Python 2.7 and BeautifoulSoup4

Install BeautifoulSoup4
```
pip install beautifulsoup4
```

### Prerequisites
